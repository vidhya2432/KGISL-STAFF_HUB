# **App Name**: AcademiaLink

## Core Features:

- Timetable Management: Upload and manage class timetables with an intuitive interface.
- Automated Notifications: Send automated notifications and reminders for classes and assignments.
- Google Drive Integration: Allow assignment uploads and submissions directly via Google Drive.
- Plagiarism Checker Tool: Use AI to detect plagiarism and duplication in student submissions. The tool may use an external plagiarism checking service when appropriate.
- Submission Tracking: Track assignment submissions and provide submission status updates.
- Performance Tracking: Track student performance and generate grade reports. (MVP includes report download in CSV or excel format only)
- AI Assignment Suggestion: Provide AI-driven suggestions for assignments based on course content.

## Style Guidelines:

- Primary color: Indigo (#4B0082) to convey knowledge, integrity, and professionalism.
- Background color: Very light lavender (#F5EEF8) to provide a gentle, calming backdrop that complements the primary color without causing distraction.
- Accent color: Violet (#8359A3) for interactive elements and highlights.
- Body and headline font: 'Alegreya', a humanist serif, is ideal for headlines or body text, to suggest elegance and readability for longer content.
- Use flat, professional icons for navigation and actions.
- Clean, structured layout with clear visual hierarchy.
- Subtle transitions and animations for a smooth user experience.
